# Valentine Invitation

React + Vite + Tailwind + GSAP 기반의 인터랙티브 발렌타인 초대장입니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview
```

## GitHub Pages 배포

1. 저장소의 기본 브랜치를 `main`으로 사용합니다.
2. GitHub 저장소 설정에서 `Settings > Pages > Build and deployment`를 `GitHub Actions`로 설정합니다.
3. `main` 브랜치로 푸시하면 `.github/workflows/deploy.yml`이 자동으로 배포합니다.
